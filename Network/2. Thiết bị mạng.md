# Báo cáo chi tiết về các thiết bị mạng

## 1. Tổng quan
Trong hệ thống mạng máy tính hiện đại, các thiết bị mạng đóng vai trò quan trọng trong việc kết nối, truyền dữ liệu và bảo mật hệ thống. Các thiết bị phổ biến nhất bao gồm:

- Switch
- Router
- Firewall
- Access Point

Mỗi thiết bị có chức năng và vai trò khác nhau trong mạng.

---

# 2. Switch

## 2.1 Mục đích
Switch được sử dụng để **kết nối nhiều thiết bị trong cùng một mạng LAN (Local Area Network)**.

Các thiết bị thường kết nối với switch:

- Máy tính (PC)
- Server
- Printer
- Camera IP
- Access Point

Ví dụ sơ đồ:

```
PC1
PC2
Server
Printer
   │
 Switch
```

Switch giúp các thiết bị trong cùng mạng **giao tiếp với nhau một cách hiệu quả**.

---

## 2.2 Nguyên lý hoạt động
Switch hoạt động dựa trên **địa chỉ MAC (MAC Address)**.

Khi một thiết bị gửi dữ liệu, switch sẽ kiểm tra bảng MAC để xác định dữ liệu cần gửi tới cổng nào.

Ví dụ bảng MAC:

| MAC Address | Port |
|---|---|
AA:BB:CC:01 | Port 1 |
AA:BB:CC:02 | Port 2 |
AA:BB:CC:03 | Port 3 |

Switch chỉ gửi dữ liệu tới đúng cổng cần thiết, giúp giảm lưu lượng mạng.

---

## 2.3 Tính năng phổ biến

- VLAN (Virtual LAN)
- STP (Spanning Tree Protocol)
- Port Trunking
- QoS (Quality of Service)
- Link Aggregation

---

## 2.4 OSI Layer

Switch hoạt động ở:

Layer 2 – Data Link

---

# 3. Router

## 3.1 Mục đích

Router được sử dụng để **kết nối các mạng khác nhau với nhau**, ví dụ:

- LAN → Internet
- LAN → LAN khác
- VLAN → VLAN

Ví dụ sơ đồ:

```
LAN Network
192.168.1.0/24
      │
    Router
      │
   Internet
```

Router quyết định **đường đi của gói tin (routing)**.

---

## 3.2 Nguyên lý hoạt động

Router hoạt động dựa trên **địa chỉ IP**.

Router sử dụng **Routing Table** để quyết định nơi gửi gói tin.

Ví dụ routing table:

```
192.168.1.0/24 → LAN
0.0.0.0/0 → Internet
```

---

## 3.3 Tính năng phổ biến

- Routing giữa các mạng
- NAT (Network Address Translation)
- DHCP Server
- VPN
- Firewall cơ bản
- Static Route
- Dynamic Routing (OSPF, BGP)

---

## 3.4 OSI Layer

Router hoạt động ở:

Layer 3 – Network

---

# 4. Firewall

## 4.1 Mục đích

Firewall dùng để **bảo vệ mạng bằng cách kiểm soát traffic ra vào mạng**.

Firewall kiểm tra các kết nối dựa trên **rule bảo mật**.

Ví dụ:

```
Internet
   │
Firewall
   │
LAN Network
```

---

## 4.2 Cách hoạt động

Firewall kiểm tra:

- IP address
- Port
- Protocol
- Application

Ví dụ rule:

Cho phép:

LAN → Internet

Chặn:

Internet → LAN

---

## 4.3 Tính năng phổ biến

- Packet filtering
- Stateful inspection
- NAT
- VPN
- IDS/IPS
- Web filtering
- Application control

---

## 4.4 Ví dụ firewall phổ biến

- Fortigate
- Palo Alto
- Cisco ASA
- Checkpoint

---

# 5. Access Point

## 5.1 Mục đích

Access Point dùng để **cung cấp kết nối WiFi cho các thiết bị không dây**.

Ví dụ:

```
Laptop
Phone
Tablet
   │
 WiFi
   │
Access Point
   │
Switch
```

Access Point kết nối **mạng không dây với mạng có dây**.

---

## 5.2 Cách hoạt động

Thiết bị kết nối vào **SSID** của Access Point.

Ví dụ SSID:

Office_Wifi  
Guest_Wifi  

Sau khi kết nối, dữ liệu sẽ được chuyển vào mạng LAN.

---

## 5.3 Tính năng phổ biến

- SSID
- WPA2 / WPA3
- Guest network
- VLAN WiFi
- Roaming
- Band steering

---

# 6. Ví dụ hệ thống mạng hoàn chỉnh

```
Internet
   │
 Router / Firewall
   │
 Switch
 ├── PC
 ├── Server
 ├── Printer
 └── Access Point
        │
      WiFi
   Laptop / Phone
```

Router: kết nối Internet  
Firewall: bảo mật hệ thống  
Switch: kết nối thiết bị LAN  
Access Point: cung cấp WiFi
