## Cấu hình Omada ER605

## 1 Kết nối phần cứng

Kết nối theo sơ đồ:

    Internet
       │
    Modem
       │
    WAN1 (ER605)
       │
    LAN Port (ER605)
       │
    PC cấu hình

Các cổng ER605:

  Port       Chức năng
  ---------- -----------------------
  WAN1       Kết nối Internet
  WAN2/LAN1   WAN/LAN
  LAN2       LAN
  LAN3       LAN

------------------------------------------------------------------------

# 3. Truy cập vào router ER605

## 3.1 Cấu hình IP cho máy tính

Đặt IP tĩnh cho PC:

    IP Address: 192.168.0.10
    Subnet Mask: 255.255.255.0
    Gateway: 192.168.0.1

## 3.2 Truy cập giao diện quản lý

Mở trình duyệt:

    http://192.168.0.1

Thông tin đăng nhập mặc định:

    Username: admin
    Password: admin

Sau khi đăng nhập cần **đổi mật khẩu quản trị**.

------------------------------------------------------------------------

# 4. Cấu hình WAN kết nối Internet

Vào menu:

    Network → WAN

Chọn kiểu kết nối:

## PPPoE (phổ biến với VNPT)

Nhập thông tin:
- Connection Type: PPPoE
- Username: ctyhpadirect
- Password:
- Service Name: Wan
- Primary DNS: 8.8.8.8
- Secondary Connection: none
Sau khi lưu router sẽ nhận **IP WAN public**.
Ví dụ:

    WAN IP: 14.224.209.208

Kiểm tra Internet:

    Status → WAN
