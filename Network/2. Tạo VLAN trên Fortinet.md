# 2. Tạo VLAN10 và VLAN20 trên port4 và port5

### Mục tiêu

-   Tạo 2 mạng nội bộ
-   Hai mạng ra Internet
-   Hai PC ping được nhau

### Cấu hình VLAN

#### VLAN10 (port4)

Network → Interfaces → Create New

  Field       Value
  ----------- ----------------
  Interface   port4
  Type        VLAN
  VLAN ID     10
  IP          172.16.10.1/24

#### VLAN20 (port5)

  Field       Value
  ----------- ----------------
  Interface   port5
  Type        VLAN
  VLAN ID     20
  IP          172.16.20.1/24

### Cấu hình IP cho PC

PC1

    IP: 172.16.10.10
    Gateway: 172.16.10.1

PC2

    IP: 172.16.20.10
    Gateway: 172.16.20.1

### Policy cho hai VLAN giao tiếp

Policy:

  Incoming   Outgoing
  ---------- ----------
  VLAN10     VLAN20

  Source   Destination
  -------- -------------
  all      all

Action: ACCEPT

Tạo thêm policy ngược lại:

  Incoming   Outgoing
  ---------- ----------
  VLAN20     VLAN10

### Kết quả

PC1 ping PC2:

    ping 172.16.20.10
