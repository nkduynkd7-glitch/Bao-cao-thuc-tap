* **Địa chỉ IP Public (WAN):** `14.224.209.208`.

## 2. Các bước thực hiện

### Bước 1: Cấu hình Web Server nội bộ (IIS)
1.  Triển khai file nội dung hiển thị tại thư mục gốc của IIS.
2.  Đổi tên file từ `helloword` thành **`index.html`** để trình duyệt tự động nhận diện trang chủ.
3.  Xác nhận nội dung hiển thị: **"Hello World"**.

### Bước 2: Cấu hình Virtual IP (VIP) trên FortiGate
Thiết lập ánh xạ cổng để chuyển hướng yêu cầu từ Internet vào máy chủ:
* **Tên VIP:** `WEB_PUBLIC`.
* **Interface:** `vnpt (port1)`.
* **Port Forwarding:** Ánh xạ cổng truy cập từ cổng **3030** sang cổng **80** của máy chủ `172.16.10.20`.

### Bước 3: Cấu hình Firewall Policy
Cho phép dữ liệu đi qua tường lửa từ vùng WAN vào máy chủ LAN:
* **Tên Policy:** `Web_public`.
* **Incoming Interface:** `virtual-wan-link`.
* **Outgoing Interface:** `VLAN10 (port4)`.
* **Destination:** Chọn Virtual IP `WEB_PUBLIC`.
* **Service:** Sử dụng cổng tùy chỉnh `Port_3030`.
* **NAT:** OFF
