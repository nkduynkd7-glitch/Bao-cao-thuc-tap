# Zalo Sheet — Chấm điểm lịch từ Zalo Group

Tự động đọc tin nhắn Zalo group, tính điểm và ghi vào Google Sheets.

## Cách hoạt động

- `phát lịch` → **+1 điểm** cho người gửi
- `bỏ lịch` → **-0.5 điểm** cho người gửi
- Tên lấy tự động từ tên Zalo người gửi
- Kết quả ghi vào 2 sheet: **Log** (raw) + **Tổng hợp** (hàng ngang per người)

## Cài đặt

### Yêu cầu
- Node.js 18+
- Tài khoản Google với Service Account
- Zalo cá nhân là thành viên group cần monitor

### Bước 1: Cài dependencies
```bash
npm install zca-js googleapis
```

### Bước 2: Cấu hình
Mở file `lich-diem.js`, sửa phần đầu:
```js
const SPREADSHEET_ID = "your-google-sheet-id";   // ID trong URL Google Sheet
const CREDENTIALS_FILE = "./credentials.json";    // Service account key file
```

### Bước 3: Google Sheets API
1. Vào [console.cloud.google.com](https://console.cloud.google.com)
2. Tạo **Service Account** → tải key JSON → lưu thành `credentials.json`
3. **Share** Google Sheet với email service account (Editor)

### Bước 4: Chạy lần đầu (scan QR)
```bash
node lich-diem.js
```
- File `qr.png` sẽ tự tạo
- Mở Zalo mobile → Quét mã QR → Xác nhận
- Session lưu vào `session.json` (không cần scan lại lần sau)

### Bước 5: Chạy thường ngày
```bash
node lich-diem.js
```

## Format tin nhắn trong group

| Gõ trong group | Kết quả |
|---|---|
| `phát lịch` | +1đ cho người gửi |
| `bỏ lịch` | -0.5đ cho người gửi |
| `hủy lịch` | -0.5đ cho người gửi |

## Files

| File | Mô tả |
|---|---|
| `lich-diem.js` | **Script chính** — chạy file này |
| `credentials.json` | Service account Google (private, không share) |
| `session.json` | Session Zalo (tự tạo sau khi scan QR) |
| `qr.png` | QR code đăng nhập (tự tạo) |

## Chạy 24/7 trên server (Linux)

```bash
npm install -g pm2
pm2 start lich-diem.js --name zalo-sheet
pm2 save
pm2 startup
```
